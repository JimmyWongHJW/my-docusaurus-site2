### 如何查单？
:::tip  
  

卡密会发送到您的邮箱，但是由于不同服务商的拦截规则问题，不保证100%能收到邮件。建议您到网站自主查询。

  
:::

主动查询方法有以下几种：

 1. 通过“订单号”查询
 2. 通过“下单邮箱”查询
